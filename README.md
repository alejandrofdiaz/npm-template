# NPM Template

## Installation

```bash
npm i --save *template-name*
```

## Scripts

> Test

```bash
npm run test
```

```bash
npm run test:watch
```

_Atención_ este comando sólo funciona si el proyecto está trackeado con GIT.

> Build

```bash
npm run build
```

> Build + Publish

```bash
npm run publish:server
```
